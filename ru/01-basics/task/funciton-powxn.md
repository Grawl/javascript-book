
# Funciton pow(x,n) 

Write a function `pow(x,n)` which returns `x` in the `n-th` power. Or, in other words, `x` is multiplied by itself `n` times.

[js]
pow(3, 2) = 3*3 = 9
pow(3, 3) = 3*3*3 = 27
pow(1, 100) = 1*1*...*1 = 1
[/js]

Create a page with prompts for `x` and `n`, then outputs the result of `pow(x,n)`.
